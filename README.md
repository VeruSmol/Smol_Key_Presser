# Smol_Key_Presser
A smol lil guy that sends F13 through F24 after a 5-second countdown.

## Bro who took the extra FN-keys??
Ever want to make a keybind in an app but there's just no good keybind that doesn't conflict?<br>
Well **_I_** did, and couldn't find one, so here's a little app to virtually press the F13-F24 keys. 

### How to use: 

1.) Download and run the included .exe _(if it gets blocked, run it anyways. Idk how to sign executables ;w;)_<br>
2.) In your "smart" app _(Logitech G-Hub, Corsair iCUE, etc etc)_ select "key press" or "keyboard shortcut" on one of your macro keys<br>
3.) In Smol_Key_Presser, select the key you want to assign, and then assign it in your "smart" app<br>
4.) In your target app _(Discord, Slack, etc)_ navigate to keybinds and start to assign a key<br>
5.) In Smol_Key_Presser, select the key you want to assign _(make it the same as your "smart" app if you want it to be linked)_<br>
6.) In your target app, begin recording the keybind before the timer ends.<br>
