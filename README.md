# Smol Key Sender

A smol lil guy that sends F13 through F24 after a 5-second countdown.

## Notable stuff

- Button-based picker for F13 through F24
- 5-second countdown before sending
- Ummmm you get to have my software on your computer

## Usage Requirements

- Windows

## Build Requirements
- Python 3.x 
- PyInstaller
- Pillow (i am too lazy to make a .ico file)
